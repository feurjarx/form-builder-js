# form-builder-js
This javascript module allows you to bind elements of the form with data.

## Use
### HTML
```HTML
<html>
<body>
  <form id="form">...</form>
  <script src="form-builder.js"></script>
  <script>
    var fb = FB.make('#form');
  </script>
</body>
</html>
```
